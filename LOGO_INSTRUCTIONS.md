# How to Add Your Logo

## Folder Location
Open this folder to add your logo:
```
C:\Users\Iain\OneDrive\Documents\sogoddamntough\assets\images\
```

## Steps

1. **Open the images folder** in Windows Explorer by pasting this path:
   ```
   C:\Users\Iain\OneDrive\Documents\sogoddamntough\assets\images
   ```

2. **Replace or add your logo** with the filename:
   ```
   logo.png
   ```

   If you want to use a different name, edit `layouts/index.html` and change:
   ```html
   <img id="headerLogo" class="header-logo" src="/images/YOUR_FILENAME.png" alt="Logo">
   ```

## Logo Requirements

- **Format**: PNG, JPG, or SVG (recommended: PNG with transparent background)
- **Size**: Keep it around 40px tall (it will scale automatically)
- **Style**: White or light colored works best on the dark background

## Preview

Your logo will appear **next to the title** "sogoddamntough.com" at the top of the page.

The site will automatically reload and show your logo!
